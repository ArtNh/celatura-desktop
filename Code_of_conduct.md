# 贡献者行为准则

作为本项目的维护者与贡献者，为了促进开放与包容的社区环境，我们承诺尊重所有提出问题、提交改进建议以及参与讨论的人员。

---

### 我们的标准

为了营造理性和建设性的交流氛围，积极的行为示例包括：

* 使用包容、友善和尊重的语言
* 尊重不同的观点与技术经验
* 理性接受建设性的批评意见
* 聚焦于对项目共同发展有利的事项
* 对社区其他成员展现同理心

不当行为示例包括：

* 使用带有性暗示的语言或图像，以及不受欢迎的关注或追求
* 恶意挑衅、侮辱性言论或人身攻击
* 公开或私下的骚扰行为
* 未经明确许可发布他人的私密信息（如物理地址或电子邮箱）
* 其他在专业环境中可被合理视为不妥当的行为

---

### 维护者责任

项目维护者有权且有责任澄清何为可接受的行为标准，并被期待对任何不可接受的行为采取适当和公正的纠正措施。

项目维护者拥有移除、编辑或拒绝不符合本行为准则的提交、Issue、Wiki 编辑及其他贡献的权力与责任。

---

### 适用范围

本行为准则适用于项目的所有公共空间，当个人代表本项目或其社区时，也适用于公共领域。

---

### 执行与反馈

如发现违反本行为准则的行为，可通过 GitHub Issues 提交报告，项目维护者将对其进行审查和调查，并采取必要的适当措施。

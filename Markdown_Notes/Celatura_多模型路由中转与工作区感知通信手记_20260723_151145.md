## 核心实现说明

1. **Rust 后端工作区自动预扫描与 Context 注入**
   - 在 `src-tauri/src/lib.rs` 中实现了 `build_workspace_context` 扫描器。
   - 当用户选中本地项目工作区时，自动扫描顶层目录结构，忽略 `node_modules`, `.git`, `target`, `.next` 等垃圾目录。
   - 提取 `package.json` / `Cargo.toml` 前 40 行核心依赖与清单，自动封装至大模型 System Prompt 中，赋予模型全局代码库感知能力。

2. **多模型 HTTP SSE 零延迟流式网络层**
   - 引入 `futures-util` 字节流解析。
   - 实现了基于 `reqwest` 的异步 SSE 客户端：针对 Gemini REST API (`streamGenerateContent`) 以及 DeepSeek / OpenAI (`/v1/chat/completions` `stream: true`)。
   - 通过 `window.emit("gemini-stream", ...)` 实时将增量 Token 广播至前端。若 API Key 未配置，平滑降级拉起系统 `gemini` CLI。

3. **前端 Chat 工作区精致联动**
   - **`src/components/ChatStreamView.tsx`**：顶部高亮常驻展示绑定的工作区路径（例如：`Celatura: D:/AI_Tools/Celatura-desktop`）。
   - 用户发送任务时禁用输入框并触发 Loading 状态，实时 `listen` 后端 Token 广播追加到 React 对话气泡，支持代码块高亮与一键复制。

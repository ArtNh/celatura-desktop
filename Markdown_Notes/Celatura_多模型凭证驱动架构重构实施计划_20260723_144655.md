## 目标与重构概述

彻底清理与系统登录、OAuth2 授权拦截、`celatura://` 深层链接相关的所有冗余逻辑。将应用回归至**模型中立、开发者优先的硬核 AI 智能体工作台**。

主要改动包含：
1. **彻底移除 OAuth 拦截流**：`page.tsx` 启动后无缝直接渲染双栏工作台页面。
2. **引入极简暗黑毛玻璃凭证设置 (`SettingsModal.tsx` & `Sidebar.tsx`)**：
   - 在侧边栏底部设置齿轮触发配置弹窗。
   - 支持 **Gemini**, **DeepSeek**, **Custom OpenAI-compatible** 多提供商 API Key 配置。
   - Rust 后端自动检测系统 `GEMINI_API_KEY` 环境变量，存在即自动点亮绿色“就绪”灯。
3. **盘活与增强大模型核心管道 (`lib.rs` & `ChatStreamView.tsx`)**：
   - 清理 `lib.rs` 中繁重的 OAuth 逻辑，新增模型配置持久化与 API Key 环境变量注入机制。
   - `ChatStreamView.tsx` 顶部 Header 引入原生多模型选择器（支持切换 Gemini Pro / Flash / DeepSeek 等）。
